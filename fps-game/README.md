\# Browser FPS Prototype



\## How to Run

1\. Upload all files to a GitHub repo

2\. Go to \*\*Settings → Pages\*\*

3\. Source: `main` branch → `/root`

4\. Open the Pages URL

5\. Click to play (fullscreen + mouse lock)



\## Controls

\- Mouse: Look / Shoot

\- Click: Fire

\- ESC: Exit fullscreen



\## Notes

\- Runs 100% client-side

\- Optimized for Chrome / Edge

\- No installs, no backend
